# 🧠 MindCera – AI Powered Emotional Wellness Assistant

> **An AI-powered mental wellness platform that provides emotion detection, empathetic conversations, personalized recommendations, crisis detection, and wellness tracking.**

MindCera is an intelligent emotional wellness assistant developed to help users understand and manage their emotional wellbeing using Artificial Intelligence, Natural Language Processing, and Machine Learning.

The platform analyzes user conversations and emotional patterns, detects potential mental health risks, recommends personalized wellness activities, and provides an interactive dashboard to monitor emotional progress over time.

---

# ✨ Features

## 🤖 AI Wellness Chatbot

* AI-powered empathetic conversations
* Context-aware responses using conversation history

* Emotion Detection
* Sentiment Analysis

## 🚨 Crisis Detection

MindCera continuously analyzes conversations for crisis indicators.

When a high-risk situation is detected, the system can:

* Generate High Risk Alerts
* Notify Guardian (Email)
* Store Alert History
* Recommend Immediate Wellness Resources

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/MindCera.git
```

---

## Backend Setup

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt
```

---

## Configure Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
DATABASE_URL=your_database_url

SECRET_KEY=your_secret_key

GEMINI_API_KEY=your_gemini_api_key

EMAIL=your_email

EMAIL_PASSWORD=your_app_password
```

---

## Run Backend

```bash
python run.py
```

Backend runs at:

```
http://127.0.0.1:8000
```

---

## Frontend

Simply open

```
frontend/index.html
```

or serve using

```
Live Server
```
